# netlify-site7

A React (Vite) navigation dashboard linking to all bhenning.com / brianhenning.com sites. Deployed to `site7.bhenning.com`.

## Tech Stack

- React 18
- Vite
- Netlify deployment

## Features

- Dashboard grid with links to all nine sites
- Dynamically resolves the base domain

## Setup

```bash
npm install
```

## Running

```bash
./run.sh
```

## Build

```bash
npm run build
```

## Deployment

Deploys to Netlify via `netlify.toml`.
